# 弹珠大师空白页问题降级测试报告

## 测试目标
验证在移除复杂 3D 模型、纹理与特效后，游戏是否可以正常加载，并保持弹珠基础物理与碰撞逻辑可用。

## 降级改动摘要
1. 移除 RoomEnvironment 环境贴图、地面噪声纹理、粒子特效实现。
2. 将弹珠材质从 `MeshPhysicalMaterial` 简化为 `MeshStandardMaterial`。
3. 仅保留并新增基础立体几何元素：
   - 绿色立方体（参考物）
   - 蓝色球体（参考物）
4. 保留核心逻辑：
   - Cannon-es 世界步进
   - 弹珠刚体创建与冲量发射
   - 球体碰撞监听
   - 进洞判定与回合切换

## 测试环境
- 前端运行：`vite` 开发服务（`http://localhost:4173`）
- 自动化：Playwright（Chromium）
- 分辨率：1440 x 900

## 测试步骤与结果
1. 启动游戏并进入“弹珠大师”页面：**成功**。
2. 点击“开始游戏”，场景加载：**成功**。
3. 基础几何体显示检查：
   - 地面：显示正常
   - 立方体：显示正常
   - 球体：显示正常
   - 玩家弹珠：显示正常
4. 基础逻辑检查（静态验证 + 运行验证）：
   - 物理步进执行：正常
   - 弹珠与坑位碰撞/进洞逻辑：代码路径保持
   - 回合 UI 显示：正常（可见“玩家的回合”）

## 性能与异常记录
- 2 秒采样 FPS：**17.94 FPS**（Playwright 容器环境）
- 页面空白：**未复现**。
- 异常现象：浏览器日志提示 Software WebGL / GPU stall 警告，属于容器图形加速限制，非业务逻辑异常。

## 结论
降级方案执行后，游戏可稳定加载并显示基础 3D 几何元素，空白页问题在本次测试中未复现。建议下一步按模块逐项恢复视觉能力（先纹理、后环境贴图、最后特效），定位触发空白页的具体资源或渲染路径。
